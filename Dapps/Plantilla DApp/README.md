`API_KEY`

## Tech Stack

**Client:** React
**Server:** Solidity
**Testing** Truffle and Ganache
**BlockChain** Polygon

## Installing packages

```shell
npm i package.json
```

## run smart contract compiler

```shell
npx truffle compile
```

## set version node with nvm

```shell
nvm use 16.19.0
```

## run react

```shell
npm start
```

## Migrar contratos a Ganache (BlockChain)

```shell
npx truffle migrate --reset
npx truffle migrate --reset --compile-all
```
